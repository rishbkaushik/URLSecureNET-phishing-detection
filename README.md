# 🔐 URLSecureNET: Intelligent Phishing Detection System

An AI-powered web application that detects whether a URL is **legitimate or phishing** using multiple Machine Learning models.

---

## 📌 Features

- 🔍 Analyze any URL instantly
- 🤖 Multiple ML models:
  - Logistic Regression
  - Naive Bayes
  - Support Vector Machine (SVM)
- 🧠 Majority Voting for final prediction
- 📊 Confidence score for predictions
- ⚠️ Detects suspicious URLs based on patterns
- 🎨 Clean and professional UI

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS
- **Backend:** Flask (Python)
- **Machine Learning:** scikit-learn
- **Deployment:** Render
- **Version Control:** Git & GitHub
